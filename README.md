## Set-up
#### How to install

1. ``git clone https://github.com/alexrah/node_basics``
2. ``cd node_basics``
2. ``npm start &``
4. ``tail -F out.log``

#### How to run & transpile & watch
1.  ``npm run watch``

#### How to build
1. ``npm run build``

#### How to siege app
1. ``npm run siege``
